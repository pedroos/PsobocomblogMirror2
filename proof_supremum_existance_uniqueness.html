ε-supremum existence and uniqueness with axiom of completeness (proof)

Mathematics

Pedro O S
May/2020

We'll try and prove with the axiom of completeness that \(A \subset \mathbb{R}\) has an ε-supremum. ε-supremum means this is a definition of supremum as an upper bound satisfying an ε-clause.

Definition: \(u \in \mathbb{R}\) is an upper bound of \(A\) if:

  • \(u \geq a, \forall a \in A\)

Definition: \(u\) is a supremum of \(A\) if:

  • \(u\) is an upper bound of \(A\)
  • \(\exists a' \in A \mid u - \varepsilon \lt a' \lt u, \forall \varepsilon \gt 0, \varepsilon \in \mathbb{R}\). (See "the ε scan" below).

Axiom of completeness:

  • Let the sets \(A, B \subset X\) such that \(X\) is ordered.
  • If \(b \geq a, \forall a \in A, \forall b \in B\), then
    • \(\exists c \in X \mid a \leq c \leq b, \forall a \in A, \forall b \in B\).

Proof steps:

Part 1

  1. Let \(A\) be any subset of \(\mathbb{R}\). Then, \(A\) is bounded from above.
  2. Let \(U\) be the set of all upper bounds of \(A\). By the definition of upper bound, \(u \geq a, \forall u \in U, \forall a \in A\).
  3. By the axiom of completeness, then \(\exists c \in \mathbb{R} \mid a \leq c \leq u, \forall a \in A, \forall u \in U\).
  4. Since \(c \geq a, \forall a \in A\), \(c\) is an upper bound of \(A\).
  5. Since \(c \leq u, \forall u \in U\), \(c\) is the lowest such upper bound. (If we were to define supremum as the lowest upper bound, this would be enough for the proof. It is not the case here. So this is only a note.)

Part 2

We verify the second clause of the definition of supremum is valid for \(c\). This must be demonstrated by contradiction. As this clause is a compound statement, it can be considered a small "sub-proof".

So, suppose the clause is not valid. First, note the clause is about the \(\varepsilon\), not about the \(a'\) (I chose to write the \(\varepsilon\) to the end of the clause in the supremum definition). Second, \(a' \in A\) is the definition or condition of existence of \(a'\), so it is not part of the predicate for \(a'\), which can be true or false.

Separating the components in the predicate for \(a'\) $$m(x) = u - \varepsilon \lt x\\n(x) = x \lt u\text{, }$$we can represent the clause by $$\forall \varepsilon \gt 0, \varepsilon \in \mathbb{R}: \exists a' \in A \mid m(a') \land n(a')$$ which can be negated by $$\neg(\forall \varepsilon \gt 0, \varepsilon \in \mathbb{R}: \exists a' \in A \mid m(a') \land n(a'))\iff\\\exists \varepsilon \gt 0, \varepsilon \in \mathbb{R} \mid (\nexists a' \in A \mid m(a') \land n(a'))\text{.}$$Note the predicate for \(a\) can be written in the form $$\nexists a' \mid p(a')\text{. }$$When the variables upon which a non-existance predicate depend can be taken to always exist, the predicate can be rewritten as a negated "for all" predicate: $$\nexists x \mid p(x) \iff \forall x : \neg p(x)$$In this case, note than in \(m, n\), all variables always exist, for the following reasons:

  • \(u\), because it is an upper bound;
  • \(\varepsilon\), because it is any real \(\gt 0\);
  • \(x = a'\), because \(a'\) is the midpoint between \(u - \varepsilon\) and \(u\), which both always exist.

So the whole clause can be rewritten as $$\exists \varepsilon \gt 0, \varepsilon \in \mathbb{R} \mid \forall a' \in A : \neg(m(a') \land n(a'))\iff\\\exists \varepsilon \gt 0, \varepsilon \in \mathbb{R} \mid \forall a' \in A : \neg m(a') \lor \neg n(a')\text{. }$$Expliciting $$\neg m(x) = u - \varepsilon \geq x\\\neg n(x) = x \geq u\text{, }$$we can write$$\exists \varepsilon \gt 0, \varepsilon \in \mathbb{R} \mid \forall a' \in A : u - \varepsilon \geq a' \lor a' \geq u\text{. }$$We must show this can't be true.

We note by the definition of upper bound, if \(u - \varepsilon \geq a', \forall a' \in A\), then \(u - \varepsilon \in U\), or \(u - \varepsilon\) is an upper bound of \(A\).

However, by the axiom of completeness, \(c = u \lte )

What this proof did

This proof identifies the sets \(A\) of real numbers and its upper bounds set \(U\) with the sets described in the Axiom of Completeness \(A\), \(B\), by verifying that by the definition of upper bound, the set \(B\) in the axiom describes the upper bound set \(U\), then implying the existence of \(c\).

If a minimum upper bound definition of supremum is maintained, we verify that \(c\) is lower than or equal to all points in the set \(A\), implying it is the minimum of that set and, as such, the minimum upper bound.

If the ε-based definition of supremum is utilized, we verify that for \(c\), the "ε-procedure" described in that definition is satisfied, because of no gaps between \(c\) and \(A\). (See "the ε scan" below.)

Uniqueness

Let's try and prove that \(A \subset X\), where \(X\) is an ordered set, has \(1\) supremum. Note: this needn't be applied to \(\mathbb{R}\).

Here are the steps taken:

  1. Suppose \(s\) and \(s_2\), supremums of \(A\), and \(U\) their upper bound set.
  2. Suppose \(s_2 \gt s\).
    • By the axiom of completeness, \(a \leq s_2, \forall a \in A\).
    • But then \(a = s_2 \gt s\), contradicting the same statement for \(s\) that \(a \leq s, \forall a \in A\).
  3. Supose \(s_2 \lt s\).
    • In an analogous way, by the axiom of completeness, \(u \geq s_2, \forall u \in U\).
    • But then \(u = s_2 \lt s\), contradicting the same statement for \(s\) that \(u \geq s, \forall u \in U\). \(\blacksquare\)

The ε scan

The ε procedure in a set \(X\) checks for points belonging to a certain subset \(Y\) inside any interval opened to the left or to the right of a certain point in \(X\).

Why "inside"? Because we're considering a dense set \(X\) so that the property that there are always points inside any interval opened inside such set is valid. So this property guarantees that we always find points by opening intervals, and then, we can just check which set they belong to, or if they belong to the set \(Y\).

In this case, we're interested in that all points found belong to our set \(Y\) which represents the set we're trying to find the supremum for, because then it means there are no points between our point of consideration (an arbitrary upper bound) and \(Y\), or no "gap", such that the point is considered to be "immediately to the side" of \(Y\).

Why look for points between two points instead of just looking at all points to the left? In this case, taking all points in a certain direction into consideration, for a proper subset \(Y\) there will certainly be points which don't belong to \(Y\) because they're "beyond" \(Y\), so to speak, and not because they're in any gap between our point an \(Y\), independent of whether the gap exists or not. So this "test" would give a "false positive" for those points. We're interested only in the points "before", not "after" the set \(Y\), so to speak.

The mid-points, by contrast, are still found "beyond" the set \(Y\), because we look for any point which may belong to \(Y\), and the set is there in the middle. Meanwhile, any points in the gap between the first point and the set, or "before" the set, won't have any mid-point belonging to the set, simply because they're on the "wrong side" of it.

Denseness supplies those mid-points.
